FUNCTIONS


A function is a scripted program designed to perform a particular task. The function takes in parameters (data) and uses the parameters to perform an output within the body of the function. The body of the function is contained within { }. Functions always return a value. In JavaScript, if no return value is specified, the function will return undefined. A console.log statement will print the result to the terminal.
Function example:

function myName(firstName, lastName){
 let fullName = firstName + lastName }
return fullName;
}console.log("John", "Doe");

OBJECTS


Objects are a container that JavaScript uses for storing various types of data. The data is stored in objects as properties which are called key:value pairs.
Here's an example of the Object structure:

var car = {type:"Ford", model:"F-150", color:"Blue"};

ARRAYS


An array is a variable which can hold more than one value at a time. If you have lists of items such as sports teams or groceries, storing the data in an array would look like this:

var team = ["Colts", "Pacers", "Indy Fuel"];
var grocery = ["apples", "oranges", grapes","bananas"]

Arrays can hold many values under a single variable. The values can be accessed by referring to an index number starting at position 0.

LOOPS


If you want to run the same code over and over again, then use a loop that can input a different value each time. There are many different kinds of loops but they basically do the same thing. The loop will run until a specified condition is met. An example of a "for loop" is as follows:

for(let i = 0; i < code.length; i ++)

CONDITIONALS


Conditional statements in code allow us to make decisions. The decision process is dependent on the different types of input. For example, if the weather is cloudy, should you bring an umbrella, wear a raincoat, or stay inside? Some conditionals can be written as "if/else", "else if", or "switch" statements. Comparison operators are usually utilized with conditional statements. Here's an example of a conditional with an "if/else" statement:


                let weather = 75;

                if(weather < 70) {
                    console.log("Wear a jacket");
                } else {
                    console.log("No jacket necessary");
                }
